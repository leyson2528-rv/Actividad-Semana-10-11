La autenticación o login permite identificar a un usuario antes de acceder a una aplicación. De esta manera se garantiza que solo las personas autorizadas puedan utilizar determinadas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes dentro del sistema. La autenticación mejora la seguridad y evita accesos no autorizados. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves API ni otros datos personales sensibles.
